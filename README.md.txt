# Análise de Vendas com IA

## Objetivo  
Explorar o uso de prompts de inteligência artificial para analisar dados de vendas e gerar insights estratégicos para o negócio.

## Estrutura do Repositório  
- `dataset/`: Contém a base de dados em Excel utilizada para análise.  
- `prompts/`: Contém os prompts criados para extrair insights da IA.  
- `insights/`: Contém as respostas e insights obtidos a partir dos prompts aplicados.

## Processo  
1. Análise da base de dados para identificar perguntas estratégicas.  
2. Criação de prompts estruturados para que a IA interprete os dados.  
3. Aplicação dos prompts para geração de insights.  
4. Documentação dos resultados no repositório.

## Principais Insights  
- Produto A teve crescimento de 50% no trimestre.  
- Campanha de marketing foi fator decisivo para o aumento das vendas.  
- Recomendações para ampliar a campanha e monitorar o desempenho em outras regiões.

## Próximos Passos  
- Automatizar a análise utilizando scripts e integração com APIs de IA.  
- Expandir o projeto para incluir análise de outros indicadores e métricas.

## Como usar este projeto  
- Faça o download da base de dados em `dataset/base_de_dados.xlsx`.  
- Leia os prompts em `prompts/prompts_analise.txt` para entender as perguntas feitas à IA.  
- Consulte os insights gerados em `insights/insights_gerados.txt`.
